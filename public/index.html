A funny and friendly bot that alerts you if it will rain 🤖

I · Overview

Climanit is a basic bot made with nodejs. It interacts with a weatherApi and post the forecast in in a twitter account.

II · Contributing

Thank you for being interested on making this bot better. We encourage everyone to help improving this project with some new features, bug fixes and performance issues.

· Contribution Guidelines

Be free to create a pull request with some feature you wish.

III · Need help?

If you need help with Climanit, feel free to open an issue with a description of the problem you're facing.
